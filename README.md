## 2. 개요
+ 명칭 : 스터디 카페 예약 프로젝트
+ 개발 기간 : 2020.12.11 ~ 2021.01.20
+ 개발 인원 : 개인 프로젝트
+ 업무 내용 :
  + 인프라구축
  + 테이블 설계
  + 회원가입 / 수정 / 탈퇴 구현
  + 로그인 / 로그아웃 구현
  + 좌석 및 사물함 예약 / 결제 / 수정 구현
  + 관리자 예약관리 페이지 구현
  + 공지사항 등록 / 조회 / 수정 / 삭제 구현
+ 개발 환경 : SpringBoot, Spring JPA ,Jdk 8, Junit5
+ 웹 서버 : Tomcat 9.0
+ 데이터 베이스 : Docker에서 MySQL 8.0 이미지 사용
+ 협업 도구 : Git
